Lucy_Light ist der IRC-Bot des Brony Radio Germanys. Die derzeit eingesetzte Version ist im Channel #brony_radio_germany auf PonyChat anzutreffen. Lucy basiert auf dem Framework Bashbot von Josh Cartwright. Funktionen werden realisiert durch PHP und Bash. Die Liste der Kommandos verwendet HTML, CSS und Bootstrap. Lucy wurde inspiriert durch PonyvilleFMs IRC-Bot Holly. Anmerkung: Antworten des Bots können um einige Sekunden verzögert gesendet werden. Bitte habe einen kleinen Moment Geduld.

Kommandos können nur noch im Channel ausgeführt werden: Lucy_Light, <command>. Ausnahme: ping funktioniert nur als Query (/QUERY Lucy_Light ping).

song

Zeigt derzeitigen Song und Sendung (falls zutreffend) an.

dj

Zeigt derzeitige (falls zutreffend) und nächste Sendung an.

vardump

Zeigt derzeitigen Song, Sendung (falls zutreffend), nächste Sendung und dj-Status an.

help

Gibt einen Link zu dieser Liste aus.

onair

Lässt eine Sendung beginnen.

offair

Lässt eine Sendung enden.

ts

Gibt die TeamSpeak-Serveradresse an.

bot

Dämlich.
